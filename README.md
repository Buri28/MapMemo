# MapMemo
できるかどうかわからないけど、手探りでマップごとにローカルメモを残すMODを制作中  

記憶力が乏しいのでマップごとに遊んだ時の課題やメモを残しておきたい

- 日本語は漢字が無理なので平仮名とかカタカナで対応予定。
